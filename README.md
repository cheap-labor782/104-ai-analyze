104 人力銀行 AI 薪資智慧預報看板

網址:https://one04-ai-analyze.onrender.com/

本專案為一個「AI 薪資精準預報與市場成效」視覺化互動看板。系統旨在透過前端介面與 AI 資料模型協作，協助使用者快速查詢特定區域、職務名稱與類別下的薪資落點預報。

專案功能
AI 即時薪資預報：使用者可透過下拉選單選擇公司區域、職務名稱及職務類別，系統將根據 AI 模型即時推算薪資落點。

互動式數據圖表：職務類別的選擇會自動觸發前端 JavaScript 函式，連動更新右側的薪資分析圖表，提供市場成效數據分析。

響應式與現代化設計：介面採用 Grid 佈局，具備良好的響應式體驗，並以橘色調作為強調色彩，提升專業感與使用者互動性。

技術棧
前端框架：原生 JavaScript, CSS, HTML5

視覺化圖表庫：ECharts (v5.4.3) - 用於呈現複雜的薪資統計與特徵權重影響圖。

核心介面模組
預報輸入區：包含三大篩選參數：

公司區域 (jobLocationAt)

職務名稱 (jobTitles)

職務類別 (jobCategory)

結果呈現區：預測薪資落點的即時顯示面板。

儀表板連結區：導向相關分析頁面或流程圖的快速按鈕。

數據圖表區：包含市場趨勢圖、數據連動分析圖以及 AI 模型特徵權重影響圖。

使用說明
進入看板頁面。

在左側表單區選擇「公司區域」、「職務名稱」與「職務類別」。

點擊「預測」按鈕（或於選擇類別後自動連動），即可在右側查看對應的市場成效分析圖表與薪資預測值。
